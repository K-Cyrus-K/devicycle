## 📖 作品概要

> **「このスマホ、みんなはどれくらい使っているんだろう？」**

**DEVICYCLE（デジクル）** は、そんな疑問を解決するガジェット特化型の資産管理・共有サービスです。
購入から故障・売却までのライフサイクルを記録し、統計データとして共有することで、ユーザーに「賢い買い替えタイミング」や「製品の耐久性」を判断する材料を提供します。

## ✨ 主な機能

こだわりの機能を4つのポイントで紹介します。

### 🔔 【実用性】スマート・デッドライン通知
資産の「権利」を守るため、期限管理を自動化しました。
* 保証終了日（30日前）や返品期限（7日前）を自動算出。
* ヘッダーのベルアイコンやマイページでタイムリーに通知します。

### 🏆 【独自性】データの物語化「デジタルの殿堂」
単なるランキングではなく、ユーザーの愛着や経験を可視化します。
* **長寿デバイス:** 1000日以上愛用された製品をピックアップ。
* **ユニークな故障理由:** 実際のユーザー体験に基づいた故障データを共有。

### ⚡ 【操作性】高度なフィルタリングとソート
ストレスのない操作性を実現しました。
* **高速な動的ソート:** JavaScriptを用い、登録順・購入日順・価格順・使用期間順に即座に並べ替え可能。
* **プライバシー管理:** デバイスごとの「公開・非公開」設定をワンタップで切り替えられます。

### 📂 【管理】インテリジェント・ファイル管理
いざという時のための「デジタル書類整理」を効率化しました。
* レシートや保証書をアップロードする際、種類に応じてサーバー側でフォルダを自動振り分け。
* アップロードしたPDFはブラウザ上で直接閲覧可能です。
